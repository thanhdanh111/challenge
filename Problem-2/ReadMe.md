##Problem 2: Fancy Form

Key Features:

1. **Real Token Data**: Fetches live token prices from the provided API endpoint
2. **Token Icons**: Uses the Switcheo token icons repository with fallback placeholders
3. **Exchange Rate Calculation**: Real-time calculation based on token prices
4. **Input Validation**: Validates amounts and shows error messages
5. **Swap Animation**: Loading states and smooth transitions
6. **Price Impact Indicator**: Shows simulated price impact with visual indicators
7. **Responsive Design**: Works well on all screen sizes
